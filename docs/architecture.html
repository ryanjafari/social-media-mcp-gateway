Why both crosspost + twitter-enhanced / bluesky-enhanced?
Crosspost does basic posting to 9 platforms through one tool. The "enhanced" servers are separate community projects that add rich read/search/engagement tools for a single platform. You can use both — crosspost for quick multi-platform blasts, enhanced servers for deeper interactions like searching tweets or browsing your Bluesky timeline.
